# 美容室（仮）HP

古民家を改装したアットホームな美容室のウェブサイト

## 概要

- **店名**: 美容室（仮）
- **コンセプト**: 古民家改装、靴を脱ぐスタイルのアットホームな美容室
- **対象**: 地域の方や知り合い向けのプライベートサロン

## 技術スタック

- **フレームワーク**: Next.js 14
- **言語**: TypeScript
- **スタイリング**: TailwindCSS
- **アイコン**: Lucide React
- **デプロイ**: Vercel

## 機能

- レスポンシブデザイン
- 古民家の温かみを感じさせるデザイン
- シンプルなページ構成
- お問い合わせ情報

## ページ構成

1. **トップページ**: ヘッダー画像と店舗紹介
2. **コンセプト**: 古民家美容室のこだわり
3. **メニュー**: サービス内容と料金
4. **アクセス**: 住所と営業時間
5. **お問い合わせ**: 連絡先情報

## 開発

```bash
# インストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build

# 本番環境起動
npm start
```

## デプロイ

このプロジェクトはVercelにデプロイされています

## カスタマイズ

- 色は `tailwind.config.js` の `kominka` テーマで管理
- フォントは Noto Serif JP を使用
- 画像は Unsplash の画像を使用（本番では実際の店舗画像に変更）
